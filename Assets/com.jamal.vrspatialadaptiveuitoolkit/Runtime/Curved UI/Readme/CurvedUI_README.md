### Overview

The `CurvedUI` and `MeshGenerator` scripts work together to procedurally generate a curved (cylindrical) 3D mesh directly inside Unity. This is highly useful for VR applications where flat UI canvases can be difficult to read at the edges. By rendering a UI Canvas onto a Render Texture and applying it to this curved mesh, you create an immersive, wrap-around interface.

- `MeshGenerator` A mathematical utility class that handles the complex polar-to-Cartesian coordinate conversions, UV mapping (including mirroring to prevent backwards text), and triangle generation for the mesh.
- `CurvedUI` A MonoBehaviour component that you attach to a GameObject in your scene. It provides a user-friendly interface to tweak the mesh parameters and updates the shape in real-time within the Editor.

### Features

- Procedural Mesh Generation Creates a perfectly smooth cylindrical slice without needing external 3D modeling software (like Blender or Maya).
- Real-Time Editor Feedback Utilizes Unity's `OnValidate` method to instantly update the mesh shape in the Scene View as you drag the sliders in the Inspector.
- Auto-Component Setup Automatically adds a `MeshFilter`, `MeshRenderer`, and attempts to load a default material from your `Resources` folder if they don't already exist.
- Corrected UV Mapping Automatically flips the horizontal UV coordinates so that when you apply a UI Render Texture, the text and buttons face the correct way and are not mirrored.

### How to Use

1. Prepare the Material Ensure you have a Material created to display your UI (typically using an Unlit shader). Place this material in a folder named `ResourcesMaterials` and name it `CurvedUIMaterial`. (Note If you don't do this, the script will still generate the mesh, but it might appear pinkmissing material).
2. Create the Object Create an empty GameObject in your Scene inside `Curved UI`. Name it `Curved Mesh`.
3. Attach Script Attach the `CurvedUI` script to this empty GameObject. The script will automatically add a `MeshFilter` and `MeshRenderer`.
4. Adjust the Curve Select the `Curved Mesh` object and look at the Inspector. Adjust the `Resolution`, `Curve Radius`, `Height`, and `Chord Angle` sliders until you get the desired screen shape.
5. Apply UI Assign the Render Texture (generated by your UI Camera rendering your Canvas) to the material attached to this mesh.

### Example Scenario

Use Case You are creating a main menu dashboard for a spaceship VR game.

1. You create an empty object and attach `CurvedUI`.
2. You want a wide, smooth screen that wraps around the player. You set the `Resolution` to `50` for a smooth curve.
3. You set `Curve Radius` to `10` (placing it a comfortable distance away), `Height` to `15`, and the `Chord Angle` to `120` to create a panoramic view.
4. As you drag the sliders, the mesh updates instantly in the scene, allowing you to perfectly align it with your spaceship cockpit model.

### Ikhtisar

Skrip `CurvedUI` dan `MeshGenerator` bekerja bersama untuk menghasilkan mesh 3D melengkung (silinder) secara prosedural langsung di dalam Unity. Hal ini sangat berguna untuk aplikasi VR di mana canvas UI datar seringkali sulit dibaca di bagian tepinya. Dengan merender UI Canvas ke sebuah Render Texture dan menerapkannya pada mesh melengkung ini, Anda dapat membuat antarmuka yang mengelilingi pengguna secara imersif.

- `MeshGenerator` Kelas utilitas matematis yang menangani konversi koordinat polar ke Cartesian yang kompleks, pemetaan UV (termasuk membalik sumbu agar teks tidak terbalik), dan pembuatan segitiga (triangles) untuk mesh.
- `CurvedUI` Komponen MonoBehaviour yang Anda pasangkan pada GameObject di scene. Skrip ini menyediakan antarmuka yang mudah digunakan untuk mengatur parameter mesh dan memperbarui bentuknya secara real-time di dalam Editor.

### Fitur Utama

- Pembuatan Mesh Prosedural Membuat potongan silinder yang sangat halus tanpa memerlukan perangkat lunak pemodelan 3D eksternal (seperti Blender atau Maya).
- Pembaruan Editor Real-Time Memanfaatkan metode `OnValidate` dari Unity untuk langsung memperbarui bentuk mesh di Scene View saat Anda menggeser slider di Inspector.
- Pengaturan Komponen Otomatis Secara otomatis menambahkan `MeshFilter`, `MeshRenderer`, dan mencoba memuat material bawaan dari folder `Resources` Anda jika belum ada.
- Pemetaan UV yang Dikoreksi Secara otomatis membalik koordinat UV horizontal sehingga ketika Anda memasukkan UI Render Texture, teks dan tombol akan menghadap ke arah yang benar dan tidak terbalik seperti cermin.

### Cara Penggunaan

1. Siapkan Material Pastikan Anda telah membuat Material untuk menampilkan UI Anda (biasanya menggunakan shader Unlit). Letakkan material ini di dalam folder bernama `ResourcesMaterials` dan beri nama `CurvedUIMaterial`. (Catatan Jika ini tidak dilakukan, skrip tetap akan membuat mesh, tetapi mungkin akan berwarna pinkkehilangan material).
2. Buat Objek Buat GameObject kosong di Scene Anda di dalam `Curved UI`. Beri nama `Curved Mesh`.
3. Pasang Skrip Pasangkan skrip `CurvedUI` pada GameObject kosong tersebut. Skrip akan secara otomatis menambahkan `MeshFilter` dan `MeshRenderer`.
4. Atur Lengkungan Pilih objek `Curved Mesh` dan lihat pada Inspector. Sesuaikan slider `Resolution`, `Curve Radius`, `Height`, dan `Chord Angle` hingga Anda mendapatkan bentuk layar yang diinginkan.
5. Terapkan UI Masukkan Render Texture (yang dihasilkan oleh UI Camera dari Canvas Anda) ke dalam material yang terpasang pada mesh ini.

### Contoh Skenario Penggunaan

Kasus Anda sedang membuat dashboard menu utama untuk game VR pesawat luar angkasa.

1. Anda membuat objek kosong dan menempelkan skrip `CurvedUI`.
2. Anda menginginkan layar lebar dan halus yang mengelilingi pemain. Anda mengatur `Resolution` ke `50` agar lengkungannya mulus.
3. Anda mengatur `Curve Radius` ke `10` (meletakkannya pada jarak yang nyaman), `Height` ke `15`, dan `Chord Angle` ke `120` untuk menciptakan tampilan panorama.
4. Saat Anda menggeser slider, mesh akan langsung diperbarui di dalam scene, memungkinkan Anda untuk menyelaraskannya dengan sempurna bersama model kokpit pesawat luar angkasa Anda.

### Inspector Variables Reference

Variable Name Type Description

---

`meshresolution` `int` Mesh grid resolution. Higher values mean smoother curvature. Resolusi grid mesh. Semakin tinggi semakin halus lengkungannya.
`curveradius` `float` Curvature radius. Small values create a sharp curve. Jari-jari lengkungan. Nilai kecil menghasilkan lengkungan yang tajam.
`height` `float` Vertical height of the UI mesh. Tinggi vertikal dari mesh UI.
`chordangle` `float` Arc chord angle. Determines how wide the UI wraps around horizontally. Sudut bentangan busur (arc). Menentukan seberapa lebar UI melingkar secara horizontal.
